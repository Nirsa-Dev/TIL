#  > 조건문
프로그램 수행 흐름을 바꾸는 역할을 하는 제어문 중 하나로 조건에 따라 다른 문장이 수행되도록 함   
#
> ### if문
if(조건식1)의 결과가 true일 경우 중괄호 안에 있는 코드를 실행   
   
```java
if(조건식1) {
  수행될 문장;
} elif(조건식2) {
} elif(조건식3) {
  수행될 문장;
} else {
  수행될 문장;
}
```

> ### swtich문
```java
switch(조건식) {
case 값1:
  수행될 문장;
  break;
case 값2:
  수행될 문장;
  break;
default:
  수행될 문장;
}
```
